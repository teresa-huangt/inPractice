background color

style = "background-color: #f03434 (blue); background-color:rgba(226, 106, 106, 0.2)(pink)"
The first background-color property will use a “safe” background color, such as a hexadecimal value, and the second background-color property will use an RGBa or HSLa value. Here, if a browser understands the RGBa or HSLa value it will render it, and if it doesn’t, it will fall back to the hexadecimal value above it.
